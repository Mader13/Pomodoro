# Cozy Pomodoro

A cozy, modern Pomodoro timer for the web. Soft colors, smooth animations, and a warm vibe to help you focus.

## Features

- Circular progress timer with smooth animations
- Three modes: Focus, Short Break, Long Break
- Customizable durations (5–60 min focus, 1–15 min short break, 5–30 min long break)
- Configurable sessions until long break (1–10)
- 4 alarm sounds to choose from
- Sound toggle
- Settings saved in localStorage
- Timer keeps running in background tabs (Web Worker)
- Responsive design

## Stack

React + TypeScript + Vite + Tailwind CSS + Motion (Framer Motion) + Lucide Icons

## Run Locally

**Prerequisites:** Node.js

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Output goes to `dist/`.
